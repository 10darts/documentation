================
API HTTP RESTful
================

**10darts** provides you with a complete **HTTP API RESTful** to access to all
the platform’s features. The base URL for all the APIs is::

    https://10darts.com

Siendo, por ejemplo, el endpoint para obtener el listado de notificaciones::

    https://10darts.com/api/v1/notifications/
